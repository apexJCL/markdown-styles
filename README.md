# Markdown Styles

## Development

1. Clone this repo
2. Run `npm install`
3. ???
4. **Profit**

NPM Commands:

+ `web`: HTTP server with style showcasing
+ `dist`: Rebuilds scss files
+ `dev`: Runs an HTTP live server with Sass file watcher

Run these commands with `npm run <command>`

## Usage

Precompiled sass files are located in `build` folder, just copy the style you would like to use.
