Markdown Styles

What's Markdown Styles

There are many users using Markdown in any kind of context, be it a personal blog, some news, documentation and so on. Also, there are a lot of people designing their own styling for the rendering part, because Markdown should be nice. So, here we have another styling library that's context-free.

How to use it?

It's as easy as importing the required css of the style that you would like to have. Thereafter, all you have to do is wrap your Markdown in a tag with markdown-styles class.
